# Maja Gedda | Personal Portfolio

A modern, elegant, and high-converting personal portfolio landing page for Maja Gedda, a Fullstack Developer and Educator. The site utilizes a multilayered parallax scrolling architecture and glassmorphism elements to showcase over 10 years of IT experience.

## The Approach: Vibe Coding with Antigravity

This project was built entirely through **"Vibe Coding"** inside Google's **Antigravity** environment! 

By leaning on AI agent collaboration, I utilized my own reference assets (brand guidelines, layout patterns, and artistic textures) alongside a powerful startup SKILLS-set provided by [skool.com/buildroom](https://skool.com/buildroom). 

The Antigravity agent ingested these skills, dynamically generated a custom `landing-page-creator` framework based on an 8-stage layout architecture, and mapped my professional resume seamlessly into a responsive, animated, Swedish-language website.

## Tech Stack
- **HTML5**: Semantic 8-stage architecture (Hero, Problem-Agitate, Value Stack, Social Proof, Transformation, Secondary CTA, Footer).
- **Vanilla CSS3**: No bloated frameworks. Utilizes robust CSS variables, Grid/Flex layouts, and global texture layers (halftone grids and dot matrixes) for premium design.
- **Vanilla JavaScript**: Lightweight Intersection Observers for `fade-in-up` scroll animations and `requestAnimationFrame` hooks for highly performant, multi-layer parallax scrolling.

## Setup & Local Development
Since this is an entirely static vanilla project, no Node.js installations or build steps are required. 

To view the project locally, run any local web server from the root directory. For example, using Python:
```bash
python3 -m http.server 8000
```
Then navigate to [http://localhost:8000/](http://localhost:8000/) in your browser.
