Work with images and textures

Working with photos, illustrations and textures is just as important as the color palette.All these components together create new meanings, which are spressed through collages, animations or static visuals. To become a clear system, the chaos of visual forms is following the main rules: the color palette is limited, and new elements must obey the main components. Photos become duochrome due to combinations of corporate colors. An accent is a bright color that is used sparingly to draw attention to key elements. It can be used for buttons, links, icons, or other interactive elements. It can also be used to highlight important information or to create a sense of hierarchy. The overall style is endured by the color palette, typography, and the way the images are processed.
