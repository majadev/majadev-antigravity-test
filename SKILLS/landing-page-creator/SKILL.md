# Landing Page Creator

## Overview

This skill provides the required architecture and sequence for creating high converting landing pages. It ensures landing pages follow proven principles for copywriting and structure to maximize conversions.

## Landing Page Architecture

Follow these principles and sequence strictly when creating landing pages:

### 1. HERO
- **Elements:** Eyebrow → Headline → Subheadline → CTA → Trust Signals
- **Job:** Create initial scroll. Grab attention immediately and communicate the core value proposition.

### 2. SUCCESS (conditional)
- **Elements:** Checkmark → Confirmation → Deliverable list
- **Job:** Kill buyer's remorse. Provide immediate reassurance and outline exactly what they get.

### 3. PROBLEM–AGITATE
- **Elements:** 3 problems with agitation → Personal transition
- **Job:** Make status quo painful. Highlight the customer's pain points before presenting the solution.

### 4. VALUE STACK
- **Elements:** 4 tiers descending → Total value → Your price
- **Job:** Make saying no feel stupid. Stack the value provided to make the price seem trivial by comparison.

### 5. SOCIAL PROOF
- **Elements:** Header → 3 testimonials with specific results
- **Job:** Let others convince them. Provide credible, specific evidence of results.

### 6. TRANSFORMATION
- **Elements:** 4 stages: Quick win → Compound → Advantage → 10x
- **Job:** Make outcome tangible. Paint a picture of the journey and the ultimate transformation the user will experience.

### 7. SECONDARY CTA
- **Elements:** Avatar stack → Question headline → "Yes" button
- **Job:** Catch the scrollers. Provide a final compelling reason for users who scrolled to the bottom to take action.

### 8. FOOTER
- **Elements:** Logo → Nav → Legal → Social
- **Job:** Professional legitimacy. Provide expected trust signals and navigation elements.
